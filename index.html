Getting started

In CRISPR/Cas9 experiments, we will perform

1) Guide RNA design 2) Repair prediction 3) Outcome analysis

There are lots of different tools that exist, here we are going to use the following web-based tools to achieve the goal.

1) Guide RNA design - CHOPCHOP

2) Repair prediction - FORECast - indelphi

3) Outcome analysis

* Sanger sequencing:  - TIDE - ICE analysis

* NGS data: - CRISPResso2 - RGENE Tools

1) pasta sequence > Click Paste Target

2) gene name from ENSEMBL gene ID, NCBI RefSeq number > Ex) slc45a2, XM_014129329.2
